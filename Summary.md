# Отчет по автоматизации

## Что реализовано

- Настроен Allure
- Настроен Docker Compose
- Найден 1 баг

## Причины расхождения с планом
- Дополнительное время ушло на настройку Docker (сложности с образами Java)
- Время на изучение структуры приложения и подбор селекторов

## Сработавшие риски
- Проблемы с Docker Compose: приложение aqa-shop запускалось быстрее, чем MySQL успевал подняться, что приводило к ошибкам подключения к БД. Решено добавлением healthcheck для MySQL и настройкой depends_on с condition: service_healthy в docker-compose.yml.
- Нестабильные тесты: устранены через явные ожидания (WebDriverWait)
